# 介绍
md-editor-v3的目录模块

# 依赖
```json
"dependencies": {
        "md-editor-v3": "^1.5.0",
        "vue": "^3.2.26"
}
```
# 属性

|  属性名   | 类型  | 描述 | 默认值|
|  ----  | ----  | ---- | ---- |
| browsingClass  | String |  正在浏览中的目录项的样式 |catalog-browsing|
| anchorOffset   | Number |   锚点偏移（用于解决fixed出现的问题） | 0|
